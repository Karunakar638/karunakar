# Your Business — Website for GitHub Pages

This repo contains the static site for Your Business.

## Files
- index.html — main page
- style.css — styling
- privacy.html — privacy policy
- terms.html — terms of service
- contact.html — contact page
- CNAME — optional (one-line with your custom domain)
- robots.txt — search engine instructions

## Deploy
1. Commit all files to the `main` branch.
2. Settings → Pages → Source: `main` branch `/ (root)`.
3. If using a custom domain, add `CNAME` and configure DNS records.

Replace `hello@yourdomain.com`, `Your Business Name`, and address placeholders with your real info.
